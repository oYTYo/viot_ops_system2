我的需求：

VideoAlarmManage.jsx定义了我系统的异常告警板块，目前我想对这个板块做出一些完善，首先是第一个：

目前的告警列表，其中倒数第二列是当前状态，最后一列是操作，目前我们系统设置的操作就是确认，点击确认之后，当前状态会从“未确认”变成“已确认”。

现在我要求你帮我完成一个优化，那就是把当前状态这一列的值，改成已处理和未处理，状态用气泡显示，点击气泡之后dropdown两个状态可以select，不需要额外一列来修改状态。



然后原本系统列表最后一列的“操作”，从原本的确认按钮改成“详情”，点击之后也是弹出一个空白的弹窗，名称叫作事件详情，内容是后续开发。

function statusClass(status) {
  // 原："已确认" -> 现："已处理"
  if (status === "已处理") return "border-[var(--color-accent)] bg-[var(--color-hover-bg)] text-[var(--color-accent)]";
  return "border-[var(--color-error-text)] bg-[var(--color-error-bg)] text-[var(--color-error-text)]";
}
